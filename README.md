# ☕ Coffee Shop Sales Dashboard

## Overview

This project focuses on analyzing coffee shop sales performance using Power BI. The dashboard provides business insights into revenue, order volume, product performance and customer purchasing patterns.

## Objectives

- Analyze overall business performance
- Track revenue trends over time
- Identify top-selling products
- Discover peak sales hours
- Support data-driven decision making

---

## Dataset

The dataset contains transaction records from a coffee shop chain, including:

- Transaction ID
- Transaction Date
- Transaction Time
- Store Location
- Product Category
- Product Type
- Quantity Sold
- Unit Price
- Revenue

---

## Data Preparation

Data was processed using Power Query:

- Removed duplicates
- Handled missing values
- Standardized data types
- Created calculated columns

---
